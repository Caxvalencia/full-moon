# SearchBox Component

Render box for searching and button for activating search.

---

## Usage

`TSX`

```tsx
<SearchBoxComponent />
```

## Properties

| placeholder |                          |
| ----------- | ------------------------ |
| Description | Placeholder of input box |
| Attribute   | placeholder              |
| Type        | `string`                 |

## Events

| Name       | Description                               |
| ---------- | ----------------------------------------- |
| `onSearch` | Emitted when the user taps on the button. |
